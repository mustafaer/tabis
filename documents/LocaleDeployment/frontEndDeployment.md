# FRONT-END DEPLOYMENT
### Linux

_(NodeJS and NPM should be installed.

https://mustafaer.net/programlama/linux-ortamlarda-node-js-kurulumu/)_

<pre>
sudo npm install -g @angular/cli@8
</pre>

<pre>
npm install
</pre>

## For Run
_(in api folder)_
<pre>
npm start
</pre>
